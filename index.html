  1. Office Suite
    e-Office ലേക്ക് ആവശ്യമായ കരടുകള്‍ എളുപ്പത്തില്‍ തയ്യാറാക്കുന്നതിനാണ് ഈ ആപ്ലിക്കേഷന്‍ വികസിപ്പിച്ചിട്ടുള്ളത്. ഇതുവഴി തയ്യാറാക്കുന്ന കരടുകള്‍ നേരിട്ട് e-Office ലേക്ക് Alignment തെറ്റാതെ പക‍ര്‍ത്താന്‍ സാധിക്കുന്നതാണ്. കൂടാതെ കരടുകള്‍ export ചെയ്ത് കമ്പ്യൂട്ടറില്‍ save ചെയ്യാനും പിന്നീട് Upload ചെയ്ത് edit ചെയ്യാവുന്നതുമാണ്. കൂടാതെ തയ്യാറാക്കിയ കരട് പ്രിന്റ് ചെയ്യുവാനും സാധിക്കുന്നതാണ്.
  2. ഓഫീസ് പ്രൊഫൈൽ അപ്‌ലോഡ് ചെയ്യുക
    ഇതിനായി ആദ്യം മാതൃക ഫയൽ (JSON) Download Sample ബട്ടൺ വഴി ഡൗൺലോഡ് ചെയ്യാുക. ഓഫീസ് വിവരങ്ങൾ പൂരിപ്പിച്ചശേഷം Upload Profile ബട്ടൺ ക്ലിക്ക് ചെയ്ത് ഫയൽ അപ്‌ലോഡ് ചെയ്യുക. ഈ വിവരങ്ങൾ പൂര്‍ണമായും browser ലാണ് save ചെയ്യപ്പെടുന്നത്. ആയതിനാല്‍ ഓരേ web browser തന്നെ ഉപയോിഗിക്കുവാന്‍ ശ്രദ്ധിക്കുക. ഇത് പ്രവര്‍ത്തിക്കുന്നതിന് internet ആവശ്യമില്ല.
  3. ഡ്രാഫ്റ്റ് തയ്യാറാക്കുക
    Letter, Proceedings, അല്ലെങ്കിൽ Circular/Notice കാർഡിലെ Launch App ബട്ടൺ ക്ലിക്ക് ചെയ്ത് ആവശ്യമുള്ള ആപ്ലിക്കേഷൻ തുറക്കുക. ഫോം ഫീൽഡുകൾ പൂരിപ്പിക്കുമ്പോൾ വലതുവശത്ത് പ്രിവ്യൂ തത്സമയം അപ്ഡേറ്റ് ആകും.
  4. Mail-merge ഉപയോഗിക്കുക
    ഒന്നിലധികം ഡ്രാഫ്റ്റുകൾ ഒരേസമയം തയ്യാറാക്കാൻ Mail Merge ഉപയോഗിക്കാം. ഇതിനായി Excel അല്ലെങ്കിൽ CSV ഫയൽ അപ്‌ലോഡ് ചെയ്യുക. Upload ചെയ്ത ഫയലില്‍ ഏറ്റവും മുകളിലായി ഒരു row വരുന്നതായി കാണാം. ഇതില്‍ {കോളം പേര്} കാണാം. ഇത് പകര്‍ത്തി ഫോം ഫീൽഡുകളിൽ {കോളം പേര്} എന്ന രൂപത്തിൽ പ്ലേസ്‌ഹോൾഡർ ചേർക്കുക. ഇപ്രകാരം വിവധ കോളം പേരുകള്‍ ആവശ്യമായ സ്ഥലങ്ങളില്‍ പകര്‍തിയ ശേഷം ഓരോ വരിയിലം ക്ലിക്ക് ചെയ്യുക. അപ്പോള്‍ {കോളം പേര്} നു പകരം ആ വരിയിലെ ഡാറ്റ വരുന്നത് കാണാം. ഇതിലെ ഓരോ കരടും e-Office ലേക്ക് പ്രത്യേകം കരടുകളായി പകര്‍ത്തുക. excel/csv ഫയലുകളിലെ ഡാറ്റ ഉപയോഗിച്ച മുഴുവന്‍ കരടുകളും pdf ആയി കാണുന്നതിനായി പ്രിവ്യൂ പാനലിലെ 📄 All Drafts ബട്ടൺ ഉപയോഗിക്കാം. ആവശ്യമെങ്കില്‍ പ്രിന്റ് എടുക്കുകയും ചെയ്യാം
  5. Logo/Header
    ലോഗോ upload ചെയ്യാം. ലോഗോ മുകളിലാണോ ഇടതു വശത്താണോ കാണപ്പെടേണ്ടത് എന്ന് തീരുമാനിക്കാം. Microsoft word ഫയലുകളില്‍ Header ചേര്‍ക്കുന്നതുപേലെ ഇതിലും Header ചേര്‍ക്കാം. ഇതിനായി jpg/png format ല്‍ Header തയ്യാറാക്കി upload ചെയ്യുക.
  6. ടെംപ്ലേറ്റ് സേവ് ചെയ്യുക
    പതിവായി ഉപയോഗിക്കുന്ന ഡ്രാഫ്റ്റ് ഒരു ടെംപ്ലേറ്റ് ആയി സേവ് ചെയ്യാം. Export ബട്ടൺ ഉപയോഗിച്ച് JSON ഫയലായി ഡൗൺലോഡ് ചെയ്ത്, അടുത്ത തവണ Import ബട്ടൺ വഴി ലോഡ് ചെയ്യുക. ഇതുവഴി ഓരോ തവണയും ആദ്യം മുതൽ ടൈപ്പ് ചെയ്യേണ്ട ആവശ്യം ഒഴിവാകും.
